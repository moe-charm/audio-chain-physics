# Audio Cable & System Chain Simulator

**「オーディオの『線』でなぜ音が変わるのか？」**  
この永遠の問いに対し、30年のオーディオベテランの知見と、最新の物理シミュレーション（RLGCモデル、誘電吸収、熱変調）を融合させて挑むシミュレーターです。

## 特徴
- **2段構えのシミュレーション**: RCAラインレベルとスピーカーパワーレベルの相互作用を個別に、かつ直列に計算。
- **物理エンジン (CableModel)**: 表皮効果、近接効果、構造（平行線/同軸）、絶縁体（被膜）の誘電損失を反映。
- **非線形エミュレーション**: 被膜による「誘電吸収（ボヤケ）」と、大電流による「熱変調（抵抗変化）」をDSPで再現。
- **可視化**: 振幅特性（dB）、位相特性（deg）に加え、時間軸での「インパルス応答」をグラフ化。
- **ターゲット**: Monarch MK3 (IEM) や Sony 最高峰ヘッドフォン等のハイエンド環境でのリケーブル・シミュレーション。
